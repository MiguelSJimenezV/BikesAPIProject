Rutas de Motos

Método Ruta Descripción
GET /api/motos Obtener todas las motos
GET /api/motos/:id Obtener una moto por ID
POST /api/motos Crear una nueva moto
PUT /api/motos/:id Actualizar una moto
DELETE /api/motos/:id Eliminar una moto
GET /api/motos/categoria/:categoriaId Obtener motos por categoría
